# Digital Marketing Course

This repository contains the source code for the Digital Marketing Course landing page project, built using Bootstrap 5.

## Features

- **Responsive Design**: Optimized for various screen sizes using Bootstrap 5 and custom media queries.
- **Modern UI**: Clean and professional layout for a digital marketing educational platform.
- **Sections**:
  - Hero Section
  - Key Highlights
  - Career Options
  - Service Offerings
  - Testimonials (if applicable)
  - Footer

## Technologies Used

- HTML5
- CSS3 (Bootstrap 5)
- SVG Icons

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Furqan15380/Digital-Marketing-Course.git
   ```
2. Open `home.html` in your web browser.
